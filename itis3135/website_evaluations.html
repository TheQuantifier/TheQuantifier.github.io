Website Evaluations

CRAP stands for Contrast, Repetition, Alignment, and Proximity—four core principles for evaluating visual design and layout. I applied them below to one well-designed site and one poorly designed site. [oai_citation:3‡oer.pressbooks.pub](https://oer.pressbooks.pub/howdyorhello/chapter/four-principles-of-document-design/?utm_source=chatgpt.com)

GOV.UK Design System (Styles)

GOV.UK’s design system exemplifies clarity and consistency across typography, spacing, color, and components. Guidance encourages predictable patterns and accessible, readable type. [oai_citation:4‡design-system.service.gov.uk](https://design-system.service.gov.uk/styles/?utm_source=chatgpt.com)

  • Contrast: Uses clear type hierarchy, generous whitespace, and distinct headings for emphasis—contrast supports scannability without relying solely on color. [oai_citation:5‡designnotes.blog.gov.uk](https://designnotes.blog.gov.uk/2015/09/16/tips-for-creating-good-typography/?utm_source=chatgpt.com)
  • Repetition: Consistent typography, spacing, and UI patterns are repeated across pages to reduce cognitive load. [oai_citation:6‡design-system.service.gov.uk](https://design-system.service.gov.uk/styles/?utm_source=chatgpt.com)
  • Alignment: Strict grid and left-aligned text blocks produce clean vertical rhythm and easy reading; components align to a common layout system. [oai_citation:7‡design.homeoffice.gov.uk](https://design.homeoffice.gov.uk/accessibility/page-structure/layout-typography?utm_source=chatgpt.com)
  • Proximity: Related elements (headings, paragraphs, lists, and controls) are grouped, with whitespace separating distinct sections, improving comprehension. [oai_citation:8‡designnotes.blog.gov.uk](https://designnotes.blog.gov.uk/2015/09/16/tips-for-creating-good-typography/?utm_source=chatgpt.com)

Ling’s Cars

Ling’s Cars is notorious for an intentionally chaotic aesthetic—overwhelming visuals, clashing elements, and motion that distracts from content (a classic “bad” example to analyze). [oai_citation:9‡Colorlib](https://colorlib.com/wp/bad-websites/?utm_source=chatgpt.com)

  • Contrast: Excessive, competing contrasts (animated banners, bright colors, competing focal points) create visual noise rather than helpful emphasis. [oai_citation:10‡Colorlib](https://colorlib.com/wp/bad-websites/?utm_source=chatgpt.com)
  • Repetition: Inconsistent styles and repeated but mismatched visual motifs feel random, reducing predictability and recognition. [oai_citation:11‡Flow Ninja](https://www.flow.ninja/blog/bad-ux-vs-good-ux-design-examples?utm_source=chatgpt.com)
  • Alignment: Busy, collage-like composition undermines a clear grid; elements feel scattered, which hurts scanning. [oai_citation:12‡Colorlib](https://colorlib.com/wp/bad-websites/?utm_source=chatgpt.com)
  • Proximity: Too many items sit close without clear grouping; content blocks bleed into each other, making it hard to know what to read first. [oai_citation:13‡Flow Ninja](https://www.flow.ninja/blog/bad-ux-vs-good-ux-design-examples?utm_source=chatgpt.com)
