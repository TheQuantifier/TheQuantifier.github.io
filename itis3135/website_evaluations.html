Website Evaluations

CRAP stands for Contrast, Repetition, Alignment, and Proximity‚Äîfour core principles for evaluating visual design and layout. I applied them below to one well-designed site and one poorly designed site. For an overview, see Four Principles of Document Design.

GOV.UK Design System (Styles)

GOV.UK‚Äôs design system exemplifies clarity and consistency across typography, spacing, color, and components. Guidance encourages predictable patterns and accessible, readable type.

  • Contrast: Uses clear type hierarchy, generous whitespace, and distinct headings for emphasis‚Äîcontrast supports scannability without relying solely on color. See Design Notes Blog.
  • Repetition: Consistent typography, spacing, and UI patterns are repeated across pages to reduce cognitive load. See GOV.UK Design System.
  • Alignment: Strict grid and left-aligned text blocks produce clean vertical rhythm and easy reading; components align to a common layout system. See Home Office Design.
  • Proximity: Related elements (headings, paragraphs, lists, and controls) are grouped, with whitespace separating distinct sections, improving comprehension. See Design Notes Blog.

Ling‚Äôs Cars

Ling‚Äôs Cars is notorious for an intentionally chaotic aesthetic‚Äîoverwhelming visuals, clashing elements, and motion that distracts from content (a classic ‚Äúbad‚Äù example to analyze). See Colorlib‚Äôs Bad Websites.

  • Contrast: Excessive, competing contrasts (animated banners, bright colors, competing focal points) create visual noise rather than helpful emphasis. Colorlib.
  • Repetition: Inconsistent styles and repeated but mismatched visual motifs feel random, reducing predictability and recognition. Flow Ninja.
  • Alignment: Busy, collage-like composition undermines a clear grid; elements feel scattered, which hurts scanning. Colorlib.
  • Proximity: Too many items sit close without clear grouping; content blocks bleed into each other, making it hard to know what to read first. Flow Ninja.
