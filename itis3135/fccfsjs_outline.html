JavaScript Curriculum Overview (Fall ’25)

Sections appear in the order shown within FCC’s “JavaScript” course website.
Section Steps / Pages Description (1–3 sentences)
Code Editors 5 steps Quick checks on editor fundamentals: files vs. projects, tabs/spaces, language modes, and extensions. Ensures you can configure an editor before writing code.
Variables and Strings 62 steps Declaring and assigning variables; string creation, interpolation, indexing, slicing, and common string methods. Emphasis on predictable naming and immutability of strings.
Booleans and Numbers 37 steps Boolean logic, truthy/falsy, arithmetic, operator precedence, and common numeric pitfalls (rounding, NaN, Infinity). Introduces comparisons and guards.
Functions 38 Function declarations vs. expressions, parameters/defaults, return values, scope/closures, and pure vs. impure functions.
Arrays 35 Indexed collections, iteration patterns, and the core toolbox: push, pop, slice, splice, includes, and mapping/filtering.
Objects 28 Key–value modeling, property access, mutation vs. cloning, and object utilities. Sets up thinking in records and dictionaries.
Loops 22 for, while, for...of, and for...in; off-by-one and termination safety. When to prefer array methods over manual loops.
JavaScript Fundamentals Review 31 Overview of JavaScript fundamentals
Higher Order Functions and Callbacks 37 Passing functions as values, composing behavior, and idioms like map/filter/reduce. Callback patterns and gotchas.
DOM Manipulation and Events 87 Selecting and updating elements, templating basics, and wiring UI behavior with the event loop and event propagation.
JavaScript and Accessibility 42 Progressive enhancement, focus management, ARIA relationships, keyboard interactions, and accessible patterns for dynamic UIs.
Debugging 8 Using the console, breakpoints, stepping, and watch expressions. Techniques to isolate logic, timing, and state bugs.
Basic Regex 47 Constructing regular expressions, character sets, quantifiers, groups, anchors, and common text-processing patterns.
Form Validation 103 Constraints, custom validators, and sanitization. Client-side feedback patterns that complement server-side checks.
Dates 5 Working with Date, parsing/formatting, time zones, and relative times. Common pitfalls and reliable comparisons.
Audio and Video Events 55 Media element APIs, event handling, and accessibility concerns for custom controls.
Maps and Sets 37 When to choose Map/Set over objects/arrays; performance considerations and common methods.
localStorage and CRUD Operations 78 Persisting small datasets, JSON serialization, and building minimal create/read/update/delete flows.
Classes 68 Class syntax, constructors, instance vs. static members, inheritance, and composition trade-offs.
Recursion 112 Thinking inductively, base/recursive cases, stack depth, and converting between loops and recursion.
Functional Programming 23 Immutability, referential transparency, and composing small pure functions for predictable code.
Asynchronous JavaScript 36 Callbacks to Promises to async/await, the event loop, microtasks, and robust async error handling.
JavaScript Review 2 Cumulative practice tying together the core ideas from the track in short exercises and mini-projects.
